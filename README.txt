GUIDE HOW TO USE FACE_REC CODE:

1. Install python-3.9.0-amd64.exe .

2. If don't have visual studio c++ install using VisualStudioSetup.exe .

3. If have NVDIA GPU install cuda_12.1.0_531.14_windows.exe.

4. Now to install all dependency library open command prompt in face recog folder
   and run this command   "pip install -r requirement.txt".

5. Now you need to add all the people face IMAGE(only jpeg/jpg and png no raw images) 
   we want to identify in folder face and all the sorce images from which we need
   to find them in data folder.

6. You are all set to run python code. Run the code in any IDE you wish you want to.
